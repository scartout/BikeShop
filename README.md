## How to use:

1. Install MySQLWorkbench. Create new empty schema "bikeshop".
2. Install Spring Tool Suite or IntelliJ IDEA Ultimate.
3. Run the application as Spring Boot App.


## Admin panel:

Look path "localhost:8080/admin".
